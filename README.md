## ✨ Overview

**Kiddo** is a cutting-edge frontend e-commerce platform that brings a sense of wonder and joy to online shopping. Built with React Native Web and Expo, it delivers an app-like experience directly in the browser with silky-smooth animations, a responsive interface, and a highly modular Server-Driven UI (SDUI) architecture.

## 🚀 Key Features

*   🎨 **Premium Aesthetic**: A vibrant color palette, ultra-smooth micro-animations, and soft "squircles" (rounded elements) that evoke a premium yet playful brand identity.
*   🛒 **Interactive Cart Drawer**: A beautifully animated sliding cart with bounce effects and real-time quantity updates. 
*   ⚡ **Server-Driven UI (SDUI)**: The entire feed is dynamically rendered based on JSON payloads, allowing for endless configurations of hero banners, product grids, and horizontal scrolling collections without changing front-end code.
*   🏃 **Blazing Fast Performance**: Powered by `@shopify/flash-list` to easily render complex lists of toys, clothes, and diapers without dropping a single frame.
*   📱 **Fully Responsive**: Carefully styled to look incredible on desktop, tablet, and mobile browsers.

## 🛠️ Technology Stack

*   **Framework:** React Native / Expo
*   **Web Target:** React Native Web
*   **State Management:** Zustand
*   **Animations:** React Native Reanimated & Lottie
*   **List Rendering:** Shopify FlashList
*   **Styling:** Custom StyleSheet API with responsive themes

## 💻 Running Locally

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/wadhwaumeshzira/Kiddo-Frontend.git
   cd Kiddo-Frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```
   Press `w` in the terminal to open the web version in your browser.

## 🏗️ Deployment

Deploying the application to Vercel is highly optimized using Expo's web export:

```bash
npm run build
```
*(This triggers `expo export -p web` which outputs a highly optimized static bundle into the `dist` folder.)*
